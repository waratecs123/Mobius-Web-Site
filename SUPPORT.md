## Support

For assistance with the Möbius platform and its applications, follow these guidelines.

### Documentation

Refer to the README.md for installation, usage, and feature details. Additional resources include the YouTube channel (https://youtube.com/@mobius_org) for tutorials and the official website (https://mobius-org.ru/) for further information.

### Reporting Issues

If you encounter bugs or have feature requests:  
- Open an issue on the GitHub repository: https://github.com/waratecs123/Mobius-Original/issues.  
- Provide a clear title, detailed description, steps to reproduce, and relevant logs or screenshots.

### Community

Join the Telegram channel (https://t.me/mobius_org) for community discussions and updates.

### Contact

For business inquiries, contact @debil_polneyshii_debil. For general support, use the Telegram channel or GitHub issues. Response times may vary.

Note: This repository is maintained on a voluntary basis; support is provided as time permits.
